# My Portfolio

A responsive personal portfolio website built with React and React Router, featuring a modern design with a warm and inviting color palette.

## ğŸ¨ Features

- **Responsive Design**: Mobile-first approach with hamburger menu for mobile devices
- **React Router v6**: Seamless navigation between pages
- **Warm Color Palette**: Bright and welcoming design
- **Four Main Sections**:
  - About: Personal introduction and professional bio
  - Projects: Showcase of development projects
  - Education: Academic background
  - Gallery: Visual portfolio gallery

## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Mukund234/my-portfolio.git
cd my-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open in your browser at `http://localhost:3000`.
